Analysis of Mobile Code

This directory contains an example of analyzing a program in the mobile ambient calculus.
The details of the analysis and the example program appear in the paper
"A Kleene Analysis of Mobile Ambients", Flemming Nielson, Hanne Riis Nielson and Shmuel Sagiv, Appeared in the proceedings of the 9th European Symposium on Programming Languages and Systems (ESOP) 2000.
More information is available in the file esop.tvp.

Run the example using the command: tvla esop
