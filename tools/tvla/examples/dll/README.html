Shape Analysis of Acyclic Doubly-Linked Lists

This directory contains an example analyses for a program that splices a new element into an unsahred acyclic doubly-linked list lists of the form 

typedef struct node {
   struct node *f;
   struct node *b;
   int data;
} *DL;

Goal

The aim of the analyses is to verify the absence of null dereferences.

Run the example with the following command: tvla splice
